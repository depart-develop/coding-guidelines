## Javascript 命名リスト

### 記法形式

|記法|記法名|例|
|----|----|----|
|コンポーネント名|パスカルケース|UserForm|
|変数名|キャメルケース|sampleFunction|
|定数名|スネークケース|API_URL|
|メソッド名|キャメルケース|addNumber|
|プロパティ名|キャメルケース|userName|
|クラス名|パスカルケース|MyCar|
|型|パスカルケース|MyType|

その他はフレームワークやライブラリの方針に沿った形での命名規則とする。
